# **App Name**: Sugar掌柜

## Core Features:

- Product Catalog: Maintain a catalog of sweets with details like name, description, price, and images.
- Inventory Management: Track stock levels of each sweet, receive alerts for low stock, and update inventory upon receiving new shipments.
- Sales and Order Tracking: Record sales transactions, generate invoices, and track order statuses from creation to delivery.
- Customer Management: Store customer information including contact details and purchase history.
- AI-Powered Trend Tool: Predict top selling product trends by geography using current inventory and sales data. Provide a geographical summary of predicted trends, and list suggested target demographics. Provide a summary for shop owners describing how and where they should focus sales. Provide shop owners with the choice of multiple predictions, as well as metrics describing certainty and validity of the trend analysis tool. 

## Style Guidelines:

- Primary color: Mellow, friendly raspberry pink (#C72B77) to evoke candy without being childish.
- Background color: Very light pinkish-grey (#F0E8E8) so the product photography stands out.
- Accent color: Slightly brighter, pinker raspberry (#D53B7D), placed to lead the eye.
- Body and headline font: 'Literata' serif to give the brand a mature, sophisticated feeling.
- Simple, clean line icons for navigation and common actions.
- Clean and intuitive layout with clear sections for each functionality.
- Subtle transitions and animations for a smooth user experience.
